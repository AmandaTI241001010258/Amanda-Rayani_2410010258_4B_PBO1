# Sistem Pengelolaan Penjualan Barang Elektronik UFO

## Deskripsi

Program ini dibuat menggunakan bahasa Java dengan konsep Pemrograman Berbasis Objek (PBO). Program digunakan untuk mengelola data barang elektronik pada sebuah toko.

Jenis produk yang tersedia:
- TV
- Kulkas
- Mesin Cuci

Program memiliki fitur:
- Tambah Produk
- Tampilkan Produk
- Cari Produk
- Edit Produk
- Hapus Produk
- Hitung Harga Setelah Diskon
